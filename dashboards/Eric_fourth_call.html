Eric Call #4 Flow — Marketing Update → Offer

Designed to resume the paused decision: check campaign progress, confirm need, then ask for a comfortable starting number and present a ramp/structure.
Call #4
Price anchor known ($10k/mo)
Goal: decision + next step
Key move: ask for starting number after context + need

Call Flow ~15–25 min

1
Re-open the loop (context)
Pick up where you left off — marketing results first.
3–5 min
Script
“Eric, last time we spoke you wanted to launch the marketing campaign, see how it performed, and then revisit this. Before we talk next steps, I’d love to hear how that’s been going.”
Ask: “What’s working so far?”
Ask: “What’s been harder than expected?”
Ask: “What did the campaign confirm for you?”
2
Confirm the need (has anything changed?)
Verify the ops/tech leadership gap still exists.
2–3 min
Script
“Given what you’re seeing so far, do you still feel like you need support on the operational and technical side — or has anything changed there?”
Ask: “What feels most urgent right now?”
3
Bridge to an offer (structure, not pressure)
Acknowledge timing/cash flow; keep recommendation intact.
1–2 min
Script
“My recommendation hasn’t changed — the full Q-Engaged partnership is still the cleanest way to support what you’re building. And I also understand timing and cash flow matter while marketing results are still coming in.”
4
Ask for his starting number
You’re not asking “budget” — you’re asking “start.”
1 min
Script (say this, then stop)
“To make this workable, what monthly number would you feel comfortable starting at right now?”
If he’s vague: “Would something closer to $5k or closer to $8k feel more realistic?”
5
Match his number to a structure
Ramp-up is the default; discount is time-bound.
4–6 min

If he says $4k–$6k → Ramp-up

“That helps. We can start there for the first month or two, then step up as the business supports it. Same partnership — we’re simply creating an on-ramp.”

If he says $6k–$7k → Time-bound bridge

“We can do that for a short, defined period, then return to standard pricing once things stabilize.”

If he’s below $4k → Downshift

“At that level I can’t deliver the leadership you need. We can do a smaller-scope consult/roadmap now, then revisit Q-Engaged when the business is ready.”

6
Close with a decision
Confirm comfort and book the next step.
2–3 min
Script
“If we went with that approach, would you be comfortable moving forward?”
If yes: “Great — I’ll send the agreement and we’ll align on start date.”
If maybe: “What would need to be true for this to be a clear yes?”

Must-cover checks Don’t skip

Tip: if Eric drifts into new territory, gently reframe:
“That’s helpful context — let’s come back to whether we should move forward and how we structure it.”

Anchor line Keep you grounded

One-liner
“I’m not changing the recommendation — I’m changing how we get there.”
Use this if you feel yourself sliding into negotiation mode. Structure is flexible; value is not.
