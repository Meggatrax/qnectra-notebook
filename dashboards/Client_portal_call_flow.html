Discovery Call Playbook — “Do I need a portal, and if yes, which kind?”

Designed for a Go-Giver call: you’re helping her decide, not selling. Context to honor: she uses Asana internally, has tried Smartsheet dashboards, and is curious what you recommend.

Mode: Help-first
Decision: Dashboard vs Guided Home Base
Output: 2–3 best-fit paths

Call Flow (25 minutes)

Use this as your backbone. You’ll branch based on what she says using the selector on the right.

1) Set intent 2) Map current flow 3) Portal worth-it test 4) Choose portal type 5) Close with recap
Opening (verbatim)
“Thanks for making time. Quick frame: I’m not here to sell you anything. You mentioned portals on my post — so the goal today is to help you decide (1) whether a portal is worth it for your model, and (2) if yes, which kind fits best. We can use Asana + Smartsheet as our baseline.”
3 questions to get oriented
  • “What do clients need in week one — a dashboard, or a guided home base?”
  • “Where do resources + replays live today?”
  • “Where do questions come in — email, Slack, Voxer, community?”
The “Portal Worth-It” Test
A portal is worth it when clarity is the constraint — usually when:
  • You re-send links/resources weekly
  • Clients ask “where do I…?” in week one
  • Replays/resources are split across tools
  • Questions come in through multiple channels
  • You want a more premium, consistent experience
If none of these are true, you likely just need a cleaner “Start Here” page.
Minimum Viable Portal Structure (tool-agnostic)
  • Start Here (Day 0–7 steps + expectations + access)
  • This Week (current focus + links + tasks)
  • Resources (single library)
  • Replays (organized by week/module)
  • Support (where to ask what + response expectations)

3 Neutral Paths (use on any call)
  • Path 1 — Optimize Smartsheet: best when she wants a dashboard (status + links + timeline).
  • Path 2 — Add a thin client-facing portal layer: keep Asana internal, portal becomes the front desk.
  • Path 3 — Coaching-first portal: best when she wants guided experience + accountability (this week, progress, assignments, replays).

“Dashboard vs Guided Home Base” (your key diagnostic)

Dashboard (Smartsheet tends to fit)
  • Status, deliverables, deadlines
  • Links to resources
  • Client sees “where things stand”
Guided Home Base (coaching-first tools tend to fit)
  • Start Here + This Week
  • Replays by week/module
  • Assignments/progress/accountability

End the call gracefully (no pitch)

Closing (verbatim)
“This was helpful. If you want, I can summarize what we discussed into a simple one-pager: the portal structure, which path fits you (Smartsheet optimize vs portal layer vs coaching-first), and 2–3 tools to consider. No pressure — it’ll just make your decision easier.”
